How to install: Pixel Path: The Road to Code

Our Github Repository
  1. Download the zip above
  2. Extract the zip into a folder
  3. This step is optional, but you can make a shortcut for the game by right-clicking "PixelPath.exe" and click create shortcut! You can move the shortcut to where you would please now.
  4. Launch the game from either the shortcut you made, or PixelPath.exe and enjoy!
